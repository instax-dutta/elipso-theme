# Elipso — Premium Dark Theme for Pterodactyl

Elipso is a Vercel-inspired dark theme for self-hosted Pterodactyl panels. It gives game server hosting providers and self-hosters a premium, professional panel experience with zero configuration.

## Positioning

- **Tagline**: Premium dark theme for self-hosted Pterodactyl game panels
- **Audience**: Self-hosters, game server providers, Minecraft server owners, Pterodactyl enthusiasts
- **Differentiator**: Vercel-caliber design at zero cost — Geist typography, near-black canvas, mesh gradients

## What It Changes

- Geist and Geist Mono typography throughout
- Dark-only token system with intentional contrast ratios
- Compact radii, restrained borders, and stronger functional hierarchy
- Gradient atmosphere on authentication pages
- Blade/admin panel overrides without a frontend rebuild

## Install

```bash
cd elipso-vercel-theme
sudo bash install.sh /var/www/pterodactyl
```

The installer creates a backup, copies theme views, CSS, and prebuilt client assets, clears Laravel caches, and fixes ownership.
